# Brand Use

The Vibe Research code is available under the MIT License, but the names
"Vibe Research", "Remote Vibes", and "BuildingHub" should not be used to imply
that an unrelated fork, installer, hosted service, catalog, or package is
official.

You may:

- say that your project is built from or compatible with Vibe Research
- fork the code and publish modified versions under a clearly different name
- link to the official repository and website

Please do not:

- publish a lookalike installer or package that appears official
- use the Vibe Research name for malware, credential collection, or deceptive
  forks
- present a third-party BuildingHub catalog as the official catalog

Official sources are listed in `SECURITY.md`.
