# Support

For normal questions, bugs, and install problems, open a GitHub issue at:

https://github.com/Clamepending/vibe-research/issues

For security-sensitive reports, follow `SECURITY.md` and avoid posting private
details publicly.

When asking for help, include:

- operating system and CPU architecture
- install command used
- output of `node --version` and `npm --version`
- output of `cd ~/.vibe-research/app && git describe --tags --always --dirty`
- the relevant section of `~/.vibe-research/server.log`, with secrets removed

Never paste API keys, provider tokens, private SSH keys, cookies, or personal
files into a public issue.
